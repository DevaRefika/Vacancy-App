Setup Instructions

Ikuti langkah-langkah di bawah ini untuk menjalankan aplikasi di lingkungan lokal Anda.
Persiapan Database (MySQL)

Karena database sudah disediakan dalam bentuk file export .sql, ikuti langkah ini:

    Pastikan MySQL (via XAMPP/Laragon) sudah menyala.

    Buka phpMyAdmin dan buat database baru dengan nama api_backend.

    Pilih database api_backend, lalu gunakan fitur Import untuk mengunggah file .sql yang telah disediakan di folder proyek.

Konfigurasi Backend (Laravel)

    Buka terminal di direktori /backend.

    Jalankan perintah untuk menginstal library yang dibutuhkan:

    composer install

Salin file konfigurasi environment:

cp .env.example .env

Konfigurasi Frontend (Next.js)

    Buka terminal baru di direktori /frontend.

    Instal dependensi Node.js:

    npm install

    Instalasi browser engine untuk keperluan pengujian (Playwright):

    npx playwright install

    Pastikan file .env.local sudah memiliki URL API yang mengarah ke Laravel:

    NEXT_PUBLIC_API_URL=http://127.0.0.1:8000/api
